ChumsLister App

ChumsLister is a desktop and mobile application in development that helps users manage and list products from various platforms such as eBay and Amazon.

This application uses the Amazon Product Advertising API to retrieve product information, including titles, images, pricing, and descriptions. The retrieved data is only used to display affiliate-linked product listings inside the application.

Users of ChumsLister will be able to browse Amazon products, view details, and purchase through affiliate links provided by the Amazon Associates Program.

This page serves as a public reference for the use of the Amazon Product Advertising API in our app.
